brundij

Run squad health checks in under three clicks!

Brundij is an open-source project for running squad health checks. It is meant to be an open-source alternative for the current squad health check tools you can find online. This also means Brundij is free and can be self hosted.

Brundij is:

  • 🔍 Open-source
  • 💸 Free
  • 🏠 Easy to self host
  • 👀 Not coupled with tracking
  • 📵 Not coupled with ads
